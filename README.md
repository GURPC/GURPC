# Green University Research and Publication Community (GURPC)

Welcome to the specialized workspace for the GURPC website development.

## Project Overview
This project aims to build a central digital hub for the GURPC community to facilitate research collaboration, knowledge sharing, and showcasing achievements.

## Documentation
*   [Detailed Project Plan](docs/GURPC_Website_Plan.md)

## Tech Stack
*   **Framework:** Next.js 14 (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **Database:** MongoDB (Planned)

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  **Build for Production:**
    ```bash
    npm run build
    npm start
    ```

## Project Structure
*   `app/`: App Router pages and layouts.
*   `components/`: Reusable UI components.
*   `lib/`: Utility functions and configuration.
*   `types/`: TypeScript type definitions.
*   `models/`: Database schemas (Mongoose).
